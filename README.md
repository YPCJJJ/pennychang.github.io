# Penny — Portfolio

一個個人作品集網站，展示 UX / UI 設計專案與研究案例。  
版型參考 [pennychang.myportfolio.com]，使用 HTML + CSS 原生開發，並透過 GitHub Pages 免費託管。

![Preview](screenshot.png)

## ✨ 功能特色
- **響應式設計**：適配桌面與行動裝置
- **作品集瀑布牆**：支援多張專案封面圖
- **Hero 區域**：清楚呈現個人標語與專長
- **頂部導覽列**：快速切換 Works 與 Resume
- **簡單好改**：只需更換圖片與文字即可更新

## 🛠 使用技術
- HTML5
- CSS3（原生，無框架）
- Google Fonts（Inter, Playfair Display）
- GitHub Pages 部署
